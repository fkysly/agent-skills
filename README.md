# Agent Skills

A collection of battle-tested agent skills for AI coding assistants.

## Available Skills

| Skill | Description |
|-------|-------------|
| [codex-review-cc](./skills/codex-review-cc/) | Code review using OpenAI Codex CLI with pitfall avoidance and network fallback |

## Install

```bash
npx skills add fkysly/agent-skills
```

Install a specific skill:

```bash
npx skills add fkysly/agent-skills --skill codex-review-cc
```
